## 프로젝트 소개 ([🎥 시연영상 보러가기](https://www.youtube.com/watch?v=yEqKRogYSWU))

> **🧑🏻‍💻 이력서를 작성하고, 피드백을 받을 수 있는 서비스**
>
> 이력, 써!는 이력서 작성을 두려워 하는 사람들에게 용기를 불어넣어주는 의미를 나타냅니다.
>
> 이력서를 작성하기 쉽게 템플릿을 제공하고, 작성한 이력서를 통해서 이력서 피드백을 받을 수 있습니다.
>
> 저희 서비스의 목표는 이력서 피드백의 기회를 제공하고, 이력서를 작성하는 사람들의 불안을 해소 시켜주는 것입니다.

### 타겟 유저

- 개발자를 희망하나 취업이 막막한 취준생들
- 이제 막 사회로 나온 졸업생들
- 이직을 희망하는 개발자들
- 이력서에 대해 도움을 주고 싶은 멘토들
- 자신이 가진 이력서 팁을 공유하고 싶은 멘토들

<h2>🌠 주요 기능</h2>

**회원**

- 멘토 / 멘티의 역할을 구분하여 회원가입
- 선택한 각 역할에 따라 입력 화면 구분

**🧑🏻‍🏫 멘토**

- 피드백 이벤트 오픈

  - 인원 제한, 이벤트 마감일, 분야 등을 지정하여 이벤트 오픈
  - 이력서 반려 사유와 함께 반려 가능

- 이벤트에 참여한 이력서 피드백
  - 이력서의 각 항목 단위로 피드백
  - 이력서 전반에 대한 총평 작성

**👩🏻‍💻 멘티**

- 피드백 이벤트 참여

  - 선착순 참여
  - 작성한 이력서 중 하나를 선택해 참여
  - 관심있는 멘토의 피드백 이벤트 오픈 소식을 이메일 알림을 통해 확인 가능

- 이력서 작성
  - 업무경험, 프로젝트 등 카테고리별로 항목을 추가하여 작성
- 이력서 관리
  - 작성한 이력서마다 메모 작성
  - 이력서 수정 및 삭제
  - 피드백 확인
- 이력서 피드백 반영
  - 피드백 코멘트 반영하여 이력서 수정
  - 이력서와 함께 각 항목에 달린 피드백 코멘트 확인
  - 수정 시 수정 이전 버전을 함께 확인 가능
  - 저장 시 피드백 반영한 새 이력서 생성
